Project Title: Enterprise Resource Management System (ERMS)

Description: 
This ERMS is a web-based platform designed to streamline management of employees, projects, and tasks.
This project features a role-based access (Admin, Manager, Employee) for security purposes.
This project will support user authentication, REST API integration, task prioritization, and reporting functionalities.

Features:
- Secure user authentication with role-based access.
- CRUD functionalities for Employees, Projects and Tasks tables.
- Web API for remote data access.
- Real-time progress tracking and reporting.
